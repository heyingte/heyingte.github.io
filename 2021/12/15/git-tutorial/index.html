Git 概述

Git 分布式版本控制工具

  • 版本控制即保留文件的所有历史版本。
  • 集中式版本控制工具：由中央服务器保存文件的所有历史版本，客户端获取最新版本后做出修改再提交至中央服务器，代表工具SVN。
  • 分布式版本控制工具：客户端可对文件进行版本控制，每个客户端都维护一个完整的版本库，一个客户端的修改可以推送给其他客户端。但在多人协作时，通常需要一个中央仓库来统一代码，代表工具Git

Git的基本工作机制如下，包含工作区，暂存区，本地库三个区域

  • 工作区：代码存放的工作目录
  • 暂存区：将代码从工作区添加到暂存区以让git进行追踪
  • 本地库：由暂存区提交至本地库，保存本地所有历史版本
